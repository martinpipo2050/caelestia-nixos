# Support

If you need help with Caelestia-NixOS, please read the documentation first.

Recommended order:

1. README
2. Getting Started
3. FAQ
4. Troubleshooting

If your issue persists, open a GitHub Issue including:

- Hardware
- NixOS version
- Error messages
- Steps to reproduce

Clear bug reports make problems much easier to solve.

Thank you for contributing to the project.

# Manifesto

> *"We don't build configurations. We build confidence."*

---

# Why This Project Exists

Caelestia-NixOS was created with a simple belief:

**Technology should empower people, not intimidate them.**

Too often, Linux, NixOS and software architecture are presented as subjects reserved for experts.

We believe the opposite.

With patience, curiosity and good documentation, anyone can learn.

---

# We Believe

We believe software should be understandable.

We believe documentation is as important as code.

We believe architecture should be simple.

We believe reproducibility matters.

We believe knowledge should be shared.

We believe curiosity is more valuable than experience.

We believe every expert was once a beginner.

We believe everyone deserves the opportunity to learn.

---

# Our Principles

## Simplicity

Simple systems are easier to understand.

Simple code is easier to maintain.

Simple documentation helps more people contribute.

Whenever possible, choose simplicity.

---

## Learning

Learning is not something that happens before building.

Learning happens by building.

Every mistake teaches something.

Every bug solved becomes knowledge.

Every improvement makes the next challenge easier.

---

## Documentation

Documentation is not an afterthought.

Documentation is part of the software.

A feature is only complete when someone else can understand it.

---

## Open Knowledge

Knowledge grows when it is shared.

This repository documents not only what was built...

...but why it was built.

Architecture.

History.

Decisions.

Mistakes.

Lessons learned.

Everything has value.

---

## Respect Upstream

Whenever possible we use official modules and upstream solutions.

Maintaining less code means spending more time improving the project.

Good engineering often means writing less, not more.

---

## Community

Everyone is welcome.

Questions are welcome.

Ideas are welcome.

Constructive criticism is welcome.

Helping others learn is one of the most valuable contributions anyone can make.

---

# Our Vision

We want Caelestia-NixOS to be more than a desktop configuration.

We want it to become a place where people discover that NixOS is understandable.

Where documentation removes fear.

Where architecture replaces confusion.

Where learning becomes enjoyable.

---
